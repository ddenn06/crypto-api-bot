# 🚀 Crypto Arbitrage & Tracker Bot

Професійний Telegram-бот для моніторингу криптовалютного ринку та пошуку арбітражних вікон.

## 🛠 Технологічний стек
- **Мова:** Python 3.x
- **Фреймворк:** aiogram 3.x (асинхронність)
- **База даних:** SQLite3 (збереження користувачів та історії спредів)
- **API Інтеграції:** Binance, WhiteBit, Bybit, KuCoin
- **Безпека:** python-dotenv (керування ключами)

## ⚡️ Функціонал
- **Курси в реальному часі:** Прямі запити до Binance API.
- **Арбітражний сканер:** Одночасний аналіз 4-х бірж для пошуку різниці в ціні (спреду).
- **Логування:** Автоматичний запис вигідних угод у базу даних.
- **Адмін-панель:** Перегляд бази користувачів прямо в чаті.
